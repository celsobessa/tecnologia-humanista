Decreto altera regras para compartilhamento de dados pela Administração Pública

2 December, 2022

O decreto n° 11.266/2022 altera o decreto n° 10.046/2019, que trata do compartilhamento de dados pela administração pública federal, que criou o Cadastro Base do Cidadão. Via newsletter InternetLab.
